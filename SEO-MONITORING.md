## Monthly SEO Checklist
- **Content Review**: Update blog posts for freshness and add new hunting season info.
- **On‑Page SEO**: Verify title tags, meta descriptions, header hierarchy, and alt text for new images.
- **Backlink Audit**: Check new inbound links and disavow low‑quality ones.
- **Technical SEO**: Run crawl (e.g., Screaming Frog) – fix broken links, duplicate content, and redirect loops.
- **Local SEO**: Ensure NAP consistency for any physical outfitter listings.
- **Schema Validation**: Verify Organization and Breadcrumb schema.
- **Performance**: Review Core Web Vitals scores.
- **Analytics Review**: Check GA4 page‑view trends and event conversions.

## Site Health Checklist
- [ ] No 4xx/5xx errors on live site (check via crawl)
- [ ] Mobile‑friendly layout (Responsive Design Test)
- [ ] SSL certificate valid (HTTPS everywhere)
- [ ] robots.txt present and correct
- [ ] sitemap.xml reachable (`/sitemap.xml` returns 200)
- [ ] No duplicate meta titles / descriptions
